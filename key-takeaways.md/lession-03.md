# Các kiến thức tổng hợp trong lesson 03

## 1. Câu điều kiện: 
- if
- if...else
- if...else if...else
- Cú pháp:
```js
if (<điều kiện>) {
// code...
}
```

## 2. Vòng lặp: for (i)
- Cú pháp vòng lặp for (i)
```js
for (<điều kiện khởi tạo>; <điều kiện lặp>; <cập nhật>) {
// code
}
```

## 3. Convention: Quy tắc: 
- snake_case (Tạm thời không dùng)
- kebab-case (đặt tên file, folder)
- camelCase (đặt tên biến)
- PascalCase (đặt tên class)o

## 4. Console log: 
- Sử dụng với nháy đơn, nháy kép
```js
console.log(‘Toi la Khanh’);
console.log(“Toi la Khanh”);
```
- Sử dụng kèm với variable
let name = “Khanh”;
```js
console.log(`Toi la ${name}`);
